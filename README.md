# Тестовый проект для ПГУТИ
- Основной сайт: localhost:8080
- phpmyadmin: localhost:8086
- Известные проблемы
  - Composer подгружает компоненты во время работы Docker, а не инициализации контейнеров
    - (При первом запуске некоторое время ошибка)
  - Vue не проверяет имена изображений
  - Не таблица, а лента изображений
  - Отображается не вся информация об изображении в карточке
  - В БД не отправляется id и имя пользователя при загрузке изображений
  - Возможно лишние поля в users таблице БД
